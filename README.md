# Comment for dependencies :
- axios for RESTFul API Interaction (mostly used for Front End) # It is useful if we want to call external API -> Think about it as API Consumption Tool
- cors (Cross-Origin Resource Sharing) helps to allow Front End calling API from different domain name # Without cors broswer will deny requests
- dotenv helps to read environment variables
- express is framework for backend Node.js # Used for creating API, process request/response, routing, middleware
- morgan helps to log HTTP Requests # Helps to debug when we develop application
- nodemon helps to restart server whenever code is changed
- winston is professional logging # Helps to save logs, categorize logs and format logs
